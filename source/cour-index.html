Tutoriel HTML

HTML est le langage de balisage standard pour les pages Web.

Avec HTML, vous pouvez créer votre propre site Web.

HTML est facile à apprendre - Vous prendrez plaisir à le faire !

Apprentissage facile avec HTML "Essayez-le vous-même"

Exemple

                            <!DOCTYPE html>
                            <html>
                            <head>
                                <title>Page Title</title>
                            </head>
                            <body>
                            
                                <h1>This is a Heading</h1>
                                <p>This is a paragraph.</p>
                            
                            </body>
                            </html>
